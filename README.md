# BinCaml-Benchmarks
A bunch of example BinCaml IL files

General structure for each example is
- Example Folder
  - example.il    (The BinCaml IL file)
  - example.c/h/o (The Example's code)
  - example.yml   (The Example expected outputs (SVCOMP has certain flags here that say what it should do / pass))
